# cotask

C++ 20 coroutine task

## Features

- Windows: IOCP

- asnyc file read
  - [x] read to buf
  - [x] read all
  - [ ] read line
- asnyc file write
  - [ ] create
  - [ ] write
  - [ ] write line
  - [ ] write append
- asnyc tcp socket
  - [x] sync listen
  - [x] sync bind
  - [x] asnyc accept
  - [x] asnyc recv once
  - [ ] asnyc recv all
  - [x] asnyc send once
  - [ ] asnyc send all
