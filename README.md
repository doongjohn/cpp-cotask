## Platform

- Windows: IOCP

## Features

- asnyc file read

  - [x] read all
  - [ ] read line
- asnyc file write
  - [ ] create
  - [ ] write
  - [ ] write line
  - [ ] write append
- asnyc tcp socket
  - [x] sync listen
  - [x] sync bind
  - [x] asnyc accept
  - [x] asnyc recv once
  - [x] asnyc recv all
  - [x] asnyc send once
  - [x] asnyc send all
- [ ] asnyc timer
- [ ] asnyc cancel
